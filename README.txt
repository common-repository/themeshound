Search engine for WordPress themes and templates.

== Description ==

Now you can search free and premium WordPress themes and template right from your WordPress Admin dashboard.

It searches in major websites like wordpress.org, themeforest.net, templatemonster.com etc.

== Installation ==

1. Upload `themeshound` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Themes Hound and start searching.
